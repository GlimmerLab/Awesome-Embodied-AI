# 强化学习

本目录收集了具身智能中强化学习相关的论文和代码实现，重点关注足式机器人和人形机器人的控制。

## 主要内容

- 四足机器人控制
- 人形机器人控制
- 仿真到实物迁移
- 动作模仿
- 策略学习
